Introduction to PyScript

The New Opening

Python is finally in our Browser

Modern HTML/CSS/JavaScript are very powerful tool to create contents, introduce innovations and communicate in the world wide web. Today I want to introduce the new alternative tool, PyScript that was developed by the Anaconda Engineers. Although it is still an early stage, this tool comprises of amazing features that will surprise you. Let's find it out!

What is PyScript?

PyScript is a front-end framework developed to integrate Python and web. Python is a prgramming language that has a fascinating and easily understood syntax with a huge community. With increasing usage of Python in the field of machine learning and data science, we were not able to use Python in the browser. This amazing PyScript aims to bring all of the features of Python into the our browser. Hence, this framework allows users to create rich Python application sin the brower using HTML's interface and the power of Pyodide, WASM, and modern web technologies.

  • Python in the browser: Enable drop-in content, external file hosting, and application hosting without the reliance on server-side configuration
  • Python ecosystem: Run many popular packages of Python and the scientific stack (such as numpy, pandas, scikit-learn, and more)
  • Python with JavaScript: Bi-directional communication between Python and Javascript objects and namespaces
  • Environment management: Allow users to define what packages and files to include for the page code to run
  • Visual application development: Use readily available curated UI components, such as buttons, containers, text boxes, and more
  • Flexible framework: A flexible framework that can be leveraged to create and share new pluggable and extensible components directly in Python

To use PyScript we now need to import the JavaScript file into our HTML document. (You can import these two lines below!) 

<link rel="stylesheet" href="https://pyscript.net/alpha/pyscript.css"/>
<script defer src="https://pyscript.net/alpha/pyscript.js"></script>

However, does this mean PyScript is running on JavaScript? Not really!. It works by Pyodide and WebAssembly. Pyodide is a Python distribution for the brower and it is a port of CPython, which is an interpreter of Python but compiled to WebAssembly. Since WebAssembly is a binary code format, browsers could also understand as well. So Python is using WebAssembly to embed a CPython interpreter in the browser and this allows Python communicate with HTML.

<py-script>
print("Hello World")
<py-script>
